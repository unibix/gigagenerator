# gigagenerator
Массовый генератор страниц с контентом для usercms используя gigachat api

## установка
1. залить файл gigagenerator.php в корень сайта
2. в начале файла указать ключ gigachat api (включая secret, не перепутайте) и пароль для прямого доступа к файлу
3. опционально (не обязательно)! в начале файла указать токен телеграм бота и айди чата куда он будет писать
